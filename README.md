
## .env.development with RabbitMQ and MongoDb Atlas URI

## Installation

```bash
$ npm install
```

## Docker with RabbitMQ
#### Need docker installed

```bash
$ docker-compose up
```

## Running the app

```bash
$ npm run start
```

## Unit Test

```bash
$ npm run test
```
